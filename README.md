# Randomized Quasi-Monte Carlo for Walk-on-Spheres (WoS)

This repository contains the code used to compute the WoS estimator for the examples shown in our two papers: ["Randomized Quasi-Monte Carlo for Walk on Spheres"](https://arxiv.org/abs/2605.12844) and ["Walk on Spheres and Array-RQMC"](https://arxiv.org/abs/2605.12844). 

## Requirements

You should have numpy, qmcpy, hilbertcurve and tqdm installed before running the code.
