# Randomized Quasi-Monte Carlo for Walk-on-Spheres 

This repository contains Python code for experiments with Walk-on-Spheres (WoS), randomized quasi-Monte Carlo (RQMC), and Array-RQMC methods.
The code compares standard Monte Carlo, standard RQMC, Array-MC, and Array-RQMC variants on several examples involving two- and three-dimensional domains presented in our two papers ["Randomized Quasi-Monte Carlo for Walk on Spheres"](https://arxiv.org/abs/2605.12844) and ["Walk on Spheres and Array-RQMC"](https://arxiv.org/abs/2605.12844). 

It also contains the code used to compute the column-wise total Sobol indices for the gasket example (`gasket_sobol_indices.py`).
